所有的编程语言都是我 ttLanguage 的子集。

aka，每个编程语言都有自己的特性，比如，java 的虚拟机，ruby 的元编程，以及过时的c的指针，js的函数式编程。

学习一个语言，要把握框架，其实学任何人类的已知知识，都和我在工地干活一样，先打地基，脚手架，框架，然后最后浇灌，装修细节。

边盖楼边装修不是现实的实践。
